# 📉 Customer Churn Analysis & Prediction

🚀 **End-to-End Data Analyst + Machine Learning Project | Python + SQL + Power BI**

---

## 📊 Project Overview

This project focuses on analyzing **customer churn behavior** and building a **machine learning model** to predict customers who are likely to leave.

The project combines **data analysis + predictive modeling + visualization** to provide actionable business insights.

---

## 🎯 Business Problem

Companies face major challenges like:

* High customer churn rates
* Difficulty identifying at-risk customers
* Loss of revenue due to poor retention strategies

👉 This project helps businesses **predict churn and improve customer retention**.

---

## 🛠️ Tech Stack

* **Python** → Pandas, NumPy, Scikit-learn
* **Machine Learning** → Classification Models (Logistic Regression / Random Forest)
* **SQL (PostgreSQL)** → Data querying
* **Power BI** → Dashboard & Visualization
